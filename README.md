# Data Analysis and ML Projects

Bu repo, veri bilimi ve makine öğrenimi süreçlerinde temel adımlar olan  
Exploratory Data Analysis (EDA) ve Machine Learning (ML) projelerimi içeren bir portfolyodur.

Bu depoda yer alan projeler hem akademik çalışmalarımın hem de profesyonel veri analizi yolculuğumun bir parçasıdır.

Bu portfolyonun temel hedefleri:

- Gerçek veri setleri üzerinde kapsamlı EDA süreçlerini sergilemek  
- Veri temizliği, eksik değer analizi, imputation tekniklerini uygulamak  
- İstatistiksel yorumlama ve veri hikâyeciliği becerilerini göstermek  
- Görselleştirme teknikleriyle içgörüleri anlaşılır kılmak  
- ML modelleri için uçtan uca pipeline kurma pratiğini göstermek  

# Proje Listesi

# Olympic EDA – 120 Years of Olympic History 

Kapsam:
- Age, Height, Weight dağılımları
- Eksik değer analizi
- KNN Imputation ile eksik Height/Weight doldurma
- Anomaly detection (uç değer analizi)
- Spor dalı bazlı fiziksel profil incelemeleri
- Madalya dağılımı ve trendler
- Yıllara göre fiziksel ve performans değişimleri

Detaylar için: `Olympic-EDA`


# Titanic EDA – Survival Analysis

Kapsam:
- Yolcu demografisi (Pclass, Sex, Age, SibSp, Parch, Fare, Embarked)
- Eksik değer analizi (Age, Cabin, Embarked, Fare)
- Kategorik ve sayısal dağılım incelemeleri
- Hayatta kalma oranlarının değişkenlere göre incelenmesi
- Korelasyon ve ilişki analizleri
- ML öncesi veri setini anlama ve hazırlama

Detaylar için: `Titanic_EDA`


# Titanic ML – ML Pipeline

Kapsam:
- Kategorik değişken kodlama (encoding)
- Train / Test ayrımı
- Farklı ML modellerinin eğitimi (ör. Logistic Regression, Random Forest)
- Confusion matrix ve metriklerle değerlendirme
- Kaggle formatında `submission.csv` üretimi

Detaylar için: `Titanic_ML`
